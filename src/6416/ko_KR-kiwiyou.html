트리인가?

문제

트리는 그래프의 일종으로, 정점이 0개이거나 아래의 조건을 만족하는 유향 그래프를 뜻한다.

  1. 정점이 1개 이상이다.
  2. 진입 간선이 없는 정점인 루트가 유일하다.
  3. 루트 노드를 제외한 모든 정점은 진입 간선이 유일하다.
  4. 루트로부터 루트를 제외한 각 정점으로 하나 이상의 간선을 거쳐 도달할 수 있는 경로가 유일하다.

예를 들어 원을 정점, 간선을 화살표로 표시한 아래 그림에서 그림 1과 그림 2는 트리이며, 그림 3은 트리가 아니다.

그림 1

그림 2

그림 3

유향 그래프의 정점과 간선의 정보를 받아 트리인지 판단해 보자.

입력

입력은 여러 개의 테스트 케이스로 이루어져 있으며, 입력의 끝에는 두 개의 정수 -1이 주어진다.

각 테스트 케이스에는 간선의 정보를 나타내는 정수들이 주어진다. 테스트 케이스의 끝에는 두 개의 정수 0이 주어진다.

간선의 정보는 정점 번호를 나타내는 정수 u와 v로 표현된다. 이는 정점 u로부터 정점 v로 가는 간선이 존재한다는 뜻이다. (1 ≤ u, v ≤ 13)

간선 정보에서 주어진 정점 외에 다른 정점은 그래프에 존재하지 않는다.

입력으로 주어지는 정수들은 하나 이상의 공백 혹은 개행 문자로 구분된다.

출력

k번째 테스트 케이스에서 주어진 그래프가 트리일 경우 Case k is a tree.를, 트리가 아닐 경우 Case k is not a tree.를 순서대로 출력한다.
